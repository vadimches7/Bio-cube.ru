# Bio-Cube.ru — аудит и правки (17.12.2025)

## Цель

Сжать главную страницу в режиме **service** (убрать дубли/перегруз), выровнять UX/CTA и убрать “заглушки”, которые ломают доверие (например, `href="#"`).

## Что изменили

### Структура главной (service vs installation)

- **`service`**: оставлено ядро без дублей  
  `Hero → Trust → Quiz → ServiceProblems → ServiceVisitChecklist → ServiceFormats → Cases → Guarantees → Team → FAQ → FinalCTA → Footer`
- **`installation`**: стройная структура “под ключ”  
  `Hero → Trust → Quiz → Cases → Process → Guarantees → Team → FAQ → FinalCTA → Footer`

Из потока `service` убраны повторяющиеся секции (они остаются в коде, но не рендерятся на главной): шаги/диагностика/разовый выезд/мини‑кейсы/до‑после/чек‑лист подготовки/Process.

### Hero (размещение и CTA)

- Главный CTA “Вызвать специалиста” — **в фирменном зелёном** (`variant="bio"`).
- Бейджи (3 плашки) перенесены **над кнопками**.
- “Листайте” переведено **в поток**, чтобы не перекрывать элементы на разных высотах экрана.
- Кнопка “Что делаем за визит” в `service` скроллит на `#service-visit`.

### Доверие/контент

- `TeamSection` убран риск “плейсхолдер‑персоналий” и непроверяемых фактов → нейтральный “Команда и подход”.
- В `CasesSection` для `service` убрано утверждение “реальные истории” → “примеры типовых ситуаций”.

### Ссылки и легальные страницы

- Убраны `href="#"` (футер, форма, финальный CTA).  
  Добавлены маршруты:
  - `/privacy`
  - `/offer`
- Телефон в финальном CTA — кликабельный `tel:`.
- Контакты вынесены в `src/lib/contact.ts` (единое место для правок).

### SEO (база)

- В `index.html` обновлены `title`, `description`, `og:title`, `og:description`.
- `og:image` временно на `/placeholder.svg` (поставить реальный OG‑баннер перед продом).

## Техническая проверка

- `npm run lint`: **0 ошибок** (есть warnings про react-refresh в UI-файлах — не блокируют).
- `npm run build`: **успешно**.
