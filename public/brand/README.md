## Brand assets (ручное размещение файлов)

Эта папка нужна для ассетов, которые не удобно хранить в `src/` и которые должны отдаваться как статические файлы.

### Фото команды для модального окна (ТОЛЬКО это окно)

Сохраните присланное фото команды как:

- `public/brand/team-modal.jpg`

Далее оно будет использоваться **только** в модалке выбора направления (`ModeSelectDialog`).

Путь настраивается в:
- `src/lib/brand.ts` → `TEAM_MODAL_PHOTO_IMAGE`

### Логотип (если понадобится)

Положите логотип как:
- `public/brand/biocube-logo.png`

Путь настраивается в:
- `src/lib/brand.ts` → `BRAND_LOGO_IMAGE`
